### Sizes

The size can be modified with the `size` property. Supported sizes are:

export const Sizes = () => ({
  components: { SProgress },
  template: `
    <div>
      <div class="a-progress-bar-wrapper">
        <s-progress size="mini" color="primary" percentage="40" />
        <code>mini</code>
      </div>
      <div class="a-progress-bar-wrapper">
        <s-progress size="default" color="secondary" percentage="60" />
        <code>default</code>
      </div>
      <div class="a-progress-bar-wrapper">
        <s-progress size="large" color="red" percentage="80" />
        <code>large</code>
      </div>
    </div>
  `
});

<br />

<Story name='Sizes'>{Sizes.bind({})}</Story>

<br />

##### Usage:

```html
<s-progress ... size="mini" />
```

### Loop

When the `infinite` property is given, the progress bar enters in a loop. In this mode, the percentage text is supressed.

export const Speed = () => ({
  components: { SProgress },
  template: `<s-progress infinite />`
});

<br />

<Story name='Speed'>{Speed.bind({})}</Story>

<br />

##### Usage:

```html
<s-progress ... infinite />
```

### Without text

When the progress percentage is not necessary, pass `hide-percentage` to omit it.

export const Text = () => ({
  components: { SProgress },
  template: `<s-progress percentage="60" hide-percentage />`
});

<br />

<Story name='Text'>{Text.bind({})}</Story>

<br />

##### Usage:

```html
<s-progress ... percentage="60" hide-percentage />
```
