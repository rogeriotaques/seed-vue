## Icons

A set of over 480 [Eva Icons](https://akveo.github.io/eva-icons) (244 outlined and 264 filled) wrapped in a component.

This component is, actually, a _wrapper_ that makes it easier to use the Eva Icons. It leverages a attributes that can be
given to every `icon` making it fit your projects, as well as, it implements some minor improvements to the animations.

Before using it, please make sure to add the following code to the `<head>` section of your pages:

```html
<link href="https://unpkg.com/eva-icons@1.1.3/style/eva-icons.css" rel="stylesheet" />
```

Alternatively, you can [download](https://github.com/akveo/eva-icons) the pack and import it from your local assets.

### Animations

There are 4 different animations supported combined with 2 styles.

Animations can be triggered on `hover` (default) as below:
