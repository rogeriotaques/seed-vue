### Sizes

The size can be modified with the `size` property. Supported sizes are:

export const Sizes = () => ({
  components: { SDot },
  template: `
    <div>
      <div class="a-dot-wrapper">
        <s-dot size="mini" color="red" />
        <code>mini</code>
      </div>
      <div class="a-dot-wrapper">
        <s-dot size="default" color="red" />
        <code>default</code>
      </div>
      <div class="a-dot-wrapper">
        <s-dot size="large" color="red" />
        <code>large</code>
      </div>
    </div>
  `
});
