# Quick start

Getting started with SEED-VUE is easy and can be done in 3 ways:

- Using `Yarn` or `NPM` install;
- Using `CDN`;
- Using `Github`;

<br />

## Setup

<br />

### With Yarn _(recommended)_

```sh
yarn add seed-vue
```

### With NPM

```sh
npm install seed-vue
```

<br />

### Using CDN

> ⚠️ This way should only be used if you are also importing `Vue` from a CDN, otherwise, it might not work!

Copy the following code and paste it near the end of your pages, before the closing `</body>` tag
and after the `<script />` tag that you are using to import `Vue`.

```html
<script src="https://unpkg.com/seed-vue@0.1.0/dist/index.js" crossorigin="anonymous"></script>
```

### Using Github

Download the necessary scripts from https://github.com/rogeriotaques/seed-vue/archive/master.zip

<br />

## Requirements

To have SEED-VUE working correctly, you must make sure your webpage is responsive by implementing the right tags.

### Use HTML5 doctype

```html
<!DOCTYPE html>
```

### Add the responsive viewport meta tag

```html
<meta name="viewport" content="width=device-width, initial-scale=1" />
```

<br />

## Import and start using it

Importing SEED-VUE can be done in 2 different ways.

### With a full import

Using a full import, all components will be immediatelly available in your app.

```js
// e.g main.js or main.ts

import Vue from 'vue';
import SeedVue from 'seed-vue'; // the components
import 'seed-vue/dist/index.css'; // the styles
import App from './YourApp.vue';

Vue.use(SeedVue);

new Vue({
  el: '#app',
  render: (h) => h(App)
});
```

### On demand (per component)

Using the on-demand import, may help reducing the footprint of your app because only the necessary components are imported.

```js
// e.g main.js or main.ts

import Vue from 'vue';
import { Button, Icon } from 'seed-vue';
import App from './YourApp.vue';

Vue.component(Button.name, Button);
Vue.component(Icon.name, Icon);

// or
// Vue.use(Button)
// Vue.use(Select)

new Vue({
  el: '#app',
  render: (h) => h(App)
});
```

### Using in the template

```html
<!-- e.g. YourApp.vue -->

<template>
  <div>
    <h1>Hello World</h1>
    <p>
      <s-button color="primary" @click="clickHandler">
        Click to say hello!
      </s-button>
    </p>
  </div>
</template>

<script>
  export default {
    name: 'YourApp',

    methods: {
      clickHandler(event) {
        console.log('CLICKED!', event);
      }
    }
  };
</script>
```
