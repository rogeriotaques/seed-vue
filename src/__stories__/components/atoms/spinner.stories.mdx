### Sizes

The size can be modified with the `size` property. Supported sizes are:

export const Sizes = () => ({
  components: { SSpinner },
  template: `
    <div>
      <div class="a-spinner-wrapper">
        <s-spinner size="mini" color="red" />
        <code>mini</code>
      </div>
      <div class="a-spinner-wrapper">
        <s-spinner size="default" color="red" />
        <code>default</code>
      </div>
      <div class="a-spinner-wrapper">
        <s-spinner size="large" color="red" />
        <code>large</code>
      </div>
    </div>
  `
});

<br />

<Story name='Sizes'>{Sizes.bind({})}</Story>

<br />

##### Usage:

```html
<s-spinner ... size="mini" />
```

### Speed

The speed can be modified with the `speed` property. Supported sizes are:

export const Speed = () => ({
  components: { SSpinner },
  template: `
    <div>
      <div class="a-spinner-wrapper">
        <s-spinner speed="default" />
        <code>default</code>
      </div>
      <div class="a-spinner-wrapper">
        <s-spinner speed="fast" />
        <code>fast</code>
      </div>
      <div class="a-spinner-wrapper">
        <s-spinner speed="furious" />
        <code>furious</code>
      </div>
    </div>
  `
});

<br />

<Story name='Speed'>{Speed.bind({})}</Story>

<br />

The animation duration for each speed is as follows:

- Default: `1.3s`
- Fast: `0.85s`
- Furious: `0.5s`

##### Usage:

```html
<s-spinner ... speed="fast" />
```

### With text

A text can be append with the `text` property.

  components: { SSpinner },
  template: `
    <div>
      <div class="a-spinner-wrapper">

        <s-spinner text="Wait a second ..." />
      </div>

  `
